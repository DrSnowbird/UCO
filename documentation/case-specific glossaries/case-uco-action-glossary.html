A
Action

A kind of UcoObject. A valid occurrence may have the following properties:

  • actionStatus at most one occurrence of ControlledVocabulary.
  • startTime at most one value of Timestamp.
  • endTime at most one value of Timestamp.
  • error any number of values of any type.
  • actionCount at most one value of NonNegativeInteger.
  • subactionRefs any number of occurrences of Action.

Something that may be done or performed.

ActionArgument

A kind of Facet. A valid occurrence satisfies the following necessary conditions:

  • name exactly one value of BasicString.
  • value exactly one value of BasicString.

A grouping of properties characterizing a single parameter for an action.

ActionLifecycle

A kind of Action. A valid occurrence satisfies the following necessary condition:

  • phaseRefs exactly one occurrence of ArrayOfAction.

A valid occurrence may also have the following properties:

  • actionStatus exactly zero occurrences of ControlledVocabulary.
  • startTime exactly zero values of Timestamp.
  • endTime exactly zero values of Timestamp.
  • (Unnamed Class) exactly zero values of any type.
  • actionCount exactly zero values of NonNegativeInteger.

An action pattern consisting of an ordered set of multiple actions or sub action-lifecycles.

ActionRefs

A kind of Facet. A valid occurrence may have the following properties:

  • environmentRef at most one occurrence of UcoObject.
  • locationRefs any number of occurrences of Location.
  • resultRefs any number of occurrences of UcoObject.
  • participantRefs any number of occurrences of UcoObject.
  • instrumentRefs any number of occurrences of UcoObject.
  • objectRefs any number of occurrences of UcoObject.
  • performerRef at most one occurrence of UcoObject.

A grouping of properties characterizing the core elements (who, how, with what, where, etc.) for an action. The properties consist of identifier references to separate UCO objects detailing the particular property.

ArrayOfAction

A kind of ArrayOfObject.

An ordered list of action object references.
