A
ArrayOfHash

A valid occurrence satisfies the following necessary condition:

  • Hash at least one occurrence of Hash.

An ordered list of hash values.

ArrayOfString

A valid occurrence may have the following properties:

  • Value any number of values of BasicString.

An ordered list of string values.

B
BuildConfigurationType

A valid occurrence may have the following properties:

  • configurationSettingDescription at most one value of BasicString.
  • configurationSettings any number of occurrences of ConfigurationSettingType.

Describes how the build utility was configured for a particular build of a particular software.

BuildInformationType

A valid occurrence may have the following properties:

  • buildID at most one value of BasicString.
  • buildProject at most one value of BasicString.
  • buildUtility at most one occurrence of BuildUtilityType.
  • buildVersion at most one value of BasicString.
  • buildLabel at most one value of BasicString.
  • Compilers any number of occurrences of CompilerType.
  • compilationDate at most one value of Timestamp.
  • buildConfiguration at most one occurrence of BuildConfigurationType.
  • buildScript at most one value of BasicString.
  • Libraries any number of occurrences of LibraryType.
  • buildOutputLog at most one value of BasicString.

Characterizes how a software was built.

BuildUtilityType

A valid occurrence satisfies the following necessary condition:

  • buildUtilityName exactly one value of BasicString.

A valid occurrence may also have the following properties:

  • SWID at most one value of BasicString.
  • CPEID at most one value of BasicString.

Identifies the utility used to build a particular software.

C
CompilerType

A valid occurrence may have the following properties:

  • compilerInformalDescription at most one value of any type.
  • SWID at most one value of BasicString.
  • CPEID at most one value of BasicString.

A compiler utilized during a particular build of a particular software.

ConfigurationSettingType

A valid occurrence satisfies the following necessary conditions:

  • itemName exactly one value of BasicString.
  • itemValue exactly one value of BasicString.

A valid occurrence may also have the following properties:

  • itemType at most one value of BasicString.
  • itemDescription at most one value of BasicString.

Describes a particular configuration setting for a tool, application or other cyber object.

ControlledDictionary

A valid occurrence satisfies the following necessary condition:

  • Entry at least one occurrence of ControlledDictionaryEntry.

A list of terms/keys and their values where the set of terms/keys is constrained to an explicitly defined set of values.

ControlledDictionaryEntry

A valid occurrence satisfies the following necessary conditions:

  • Key exactly one occurrence of ControlledVocabulary.
  • Value exactly one value of BasicString.

A single term/key and its value where the term/key is constrained to an explicitly defined set of values.

D
DependencyType

A valid occurrence satisfies the following necessary condition:

  • dependencyDescription exactly one value of any type.

A valid occurrence may also have the following properties:

  • dependencyType at most one value of BasicString.

Describes a single dependency for a tool.

Dictionary

A valid occurrence satisfies the following necessary condition:

  • Entry at least one occurrence of DictionaryEntry.

A list of terms/keys and their values.

DictionaryEntry

A valid occurrence satisfies the following necessary conditions:

  • Key exactly one value of BasicString.
  • Value exactly one value of BasicString.

A single of term/key and its value.

E
ExecutionEnvironmentType

A valid occurrence may have the following properties:

  • computerSpecificationRefs any number of occurrences of CyberItem.
  • userAccountRef at most one occurrence of CyberItem.
  • commandLine at most one value of BasicString.
  • startTime any number of values of Timestamp.

Describes the execution environment of a particular software execution.

F
Facet

A grouping of properties characterizing a particular aspect/facet of an object.

H
Hash

A valid occurrence satisfies the following necessary conditions:

  • hashMethod exactly one occurrence of ControlledVocabulary.
  • hashValue exactly one value of HexBinary.

A cryptographic hash of data.

L
LibraryType

A valid occurrence satisfies the following necessary conditions:

  • libraryName exactly one value of BasicString.
  • libraryVersion exactly one value of BasicString.

A library incorporated into the build of a software.
